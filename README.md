# Seleção de Cursos com Radio Button

Card interativo para escolher planos/cursos usando radio buttons customizados em HTML e CSS.

Este projeto é uma demonstração estática feita com **HTML e CSS**, pensada para funcionar diretamente no navegador e poder ser publicada com **GitHub Pages**.

## Funcionalidades

- radio buttons ocultos
- labels clicáveis com estado selecionado
- destaque roxo no item escolhido
- opções HTML & CSS, JavaScript e PHP & MySQL

## Estrutura do projeto

- `css/style.css`
- `img/icone.png`
- `index.html`

## Visual

O projeto usa um card branco sobre fundo roxo com destaque roxo claro.

## Como executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` diretamente no navegador.
3. Se preferir usar um servidor local, execute na pasta do projeto:

```bash
python -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000
```

## GitHub Pages

Como este projeto é estático, ele pode ser publicado no GitHub Pages usando a branch principal.

Configuração recomendada:

- Branch: `main`
- Pasta: `/ (root)`
